Installation
=======

System Requirements
-------

You need:

- **PHP >= 7.0** but the latest stable version of PHP is recommended
- the `mbstring` extension
- the `intl` extension

Installation
--------

~~~
$ composer require league/uri-components
~~~

Dependencies
-------

- [PHP Domain Parser](https://github.com/jeremykendall/php-domain-parser)
